# 游戏礼包弹窗式激励广告 - 新增功能描述文档

## 一、功能概述

### 1.1 功能名称
**游戏礼包弹窗式激励广告**

### 1.2 功能描述
在应用中实现一个模仿"游戏礼包"样式的弹窗对话框，用户点击"立即领取"按钮后，播放激励视频广告，观看完整视频后发放游戏奖励。该功能将游戏化设计理念与激励广告结合，提升用户观看意愿和广告转化率。

### 1.3 使用场景
- 游戏通关后发放奖励
- 签到完成领取礼包
- 任务达成奖励发放
- 每日登录奖励
- 等级提升奖励

#### 4.2.2 新增方法

**方法1: loadRewardAd() - 预加载激励广告**

**功能**: 在用户可能触发礼包弹窗前，提前加载激励广告，实现秒开体验。

**实现逻辑**:
```typescript
async loadRewardAd(): Promise<void> {
  // 1. 创建广告加载监听器
  const adLoadListener: advertising.AdLoadListener = {
    onAdLoadFailure: (errorCode, errorMsg) => {
      // 加载失败处理
    },
    onAdLoadSuccess: (ads) => {
      // 加载成功，保存广告对象
      this.rewardAd = ads[0];
    }
  };
  
  // 2. 创建AdLoader对象
  const adLoader: advertising.AdLoader = new advertising.AdLoader(this.context);
  
  // 3. 构建请求参数
  const adRequestParams: advertising.AdRequestParams = {
    adId: this.rewardAdUnitId,
    adType: AdType.REWARD
  };
  
  // 4. 发起加载请求
  adLoader.loadAd(adRequestParams, this.adOptions, adLoadListener);
}
```

**使用时机**:
- 页面初始化时（aboutToAppear）
- 进入可能触发弹窗的场景前

---

**方法2: showRewardAd() - 展示激励广告**

**功能**: 展示预加载的激励广告，并通过回调返回奖励结果。

**参数**:
- `callback: (reward: advertising.RewardAdResult) => void` - 奖励回调函数

**实现逻辑**:
```typescript
async showRewardAd(callback: (reward: advertising.RewardAdResult) => void): Promise<void> {
  // 1. 检查广告是否已加载
  if (!this.rewardAd) {
    callback({ isValid: false, amount: 0, rewardName: '' });
    return;
  }
  
  // 2. 注册状态监听器
  new RewardAdStatusHandler().registerPPSReceiver();
  
  // 3. 设置奖励回调
  this.rewardAd.on('reward', (rewardInfo: advertising.RewardAdResult) => {
    callback(rewardInfo);  // 将奖励信息传递给调用方
  });
  
  // 4. 展示广告
  advertising.showAd(this.rewardAd, this.adDisplayOptions, this.context);
}
```

**回调数据结构**:
```typescript
interface RewardAdResult {
  isValid: boolean;      // 奖励是否有效
  amount: number;        // 奖励数量
  rewardName: string;    // 奖励名称
}
```

## 九、常见问题

### 9.1 广告未加载
**问题**: 点击领取按钮后提示"激励广告未预加载"  
**解决**: 确保在aboutToAppear中调用了loadRewardAd()

### 9.2 弹窗无法打开
**问题**: dialogController.open()无效  
**解决**: 检查CustomDialogController是否正确初始化

### 9.3 奖励未发放
**问题**: 看完广告但未收到奖励通知  
**解决**: 
- 检查callback是否正确设置
- 查看日志中的reward事件是否触发

### 9.4 广告位ID错误
**问题**: 正式环境广告加载失败  
**解决**: 将testx9dtjwj8hp替换为正式广告位ID
