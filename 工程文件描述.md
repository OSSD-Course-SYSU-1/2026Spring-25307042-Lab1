# Video-Show-Master 工程文件描述文档

## 一、项目概述

### 1.1 基本信息
- **项目名称**: video-show-master
- **项目类型**: HarmonyOS原生应用（前端应用）
- **项目描述**: 基于Video组件实现长视频播放功能的HarmonyOS应用示例
- **技术栈**: HarmonyOS + ArkTS + ArkUI
- **目标平台**: HarmonyOS 6.0.2 Release及以上
- **开发工具**: DevEco Studio 6.0.2 Release及以上
- **许可证**: Apache 2.0

### 1.2 项目特点
- 完整的视频播放功能实现
- 模块化、组件化架构设计
- 与系统播控中心深度集成
- 支持后台播放和小窗播放
- 良好的用户体验优化

## 四、源代码文件详解

### 4.1 入口文件

#### EntryAbility.ets
**文件路径**: `entry/src/main/ets/entryability/EntryAbility.ets`  
**作用**: 应用主入口类  
**功能描述**:
- 继承自UIAbility
- 初始化应用上下文
- 设置深色模式
- 窗口全屏布局
- 加载主页面

**关键代码**:
```typescript
export default class EntryAbility extends UIAbility {
    onCreate(want: Want, launchParam: AbilityConstant.LaunchParam): void
    onWindowStageCreate(windowStage: window.WindowStage): void
}
```

### 4.2 页面文件

#### Index.ets
**文件路径**: `entry/src/main/ets/pages/Index.ets`  
**作用**: 主页面，核心业务逻辑  
**功能描述**:
- 视频播放控制（播放、暂停、停止）
- 全屏/非全屏切换
- 播放进度控制
- 播放速度调节（1.0X, 1.25X, 1.75X, 2.0X）
- 音量调节
- 静音播放
- 循环播放
- 小窗播放
- AVSession播控中心集成
- 长按倍速播放

**核心组件**:
- Video组件：视频播放核心
- Slider组件：播放进度条
- Menu/MenuItem组件：倍速选择菜单

### 4.3 常量定义

#### CommonConstants.ets
**文件路径**: `entry/src/main/ets/constants/CommonConstants.ets`  
**作用**: 通用常量定义  
**主要内容**:
```typescript
export class CommonConstants {
    static readonly MIN_VOLUME: number = 0.0      // 最小音量
    static readonly MAX_VOLUME: number = 15.0     // 最大音量
    static readonly COMPONENT_SPACE: number = 12  // 组件间距
    static readonly PADDING_VALUE: number = 16    // 内边距值
}
```

#### VideoStatus.ets
**文件路径**: `entry/src/main/ets/constants/VideoStatus.ets`  
**作用**: 视频状态枚举  
**主要内容**:
```typescript
export enum VideoStatus {
    PLAY_PREPARED,   // 准备完成
    PLAY_STARTED,    // 开始播放
    PLAY_PAUSED,     // 暂停
    PLAY_FINISHED,   // 播放完成
    PLAY_STOPPED     // 停止
}
```

### 4.4 数据模型

#### VideoInfo.ets
**文件路径**: `entry/src/main/ets/module/VideoInfo.ets`  
**作用**: 视频信息接口定义  
**主要内容**:
```typescript
export interface VideoInfo {
    name: Resource,          // 视频名称
    index: number,          // 索引
    videoSrc: Resource,     // 视频源
    previewUri: Resource,   // 预览图
    description: Resource   // 描述

```

#### VideoData.ets
**文件路径**: `entry/src/main/ets/module/VideoData.ets`  
**作用**: 视频数据管理

### 4.5 控制器

#### AVSessionController.ets
**文件路径**: `entry/src/main/ets/controller/AVSessionController.ets`  
**作用**: AVSession控制器，管理媒体会话  
**功能描述**:
- 创建和管理AVSession会话
- 设置媒体元数据
- 设置播放状态
- 监听播控中心事件（播放、暂停、跳转、上一个/下一个）
- 后台任务管理

**关键方法**:
```typescript
createAVSession(): void              // 创建AVSession会话
setAVSessionMetadata(): void         // 设置媒体元数据
setAVSessionPlaybackState(): void    // 设置播放状态
registerAVSessionCallback(): void    // 注册回调监听
```
